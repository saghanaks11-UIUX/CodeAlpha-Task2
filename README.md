# CodeAlpha UI/UX Internship
## Task 2: High-Fidelity UI Design

### Project Title
Bookspace – E-Book Reading Mobile App (High-Fidelity UI Design)

---

## Project Overview
This project was created as part of the UI/UX Design Internship at CodeAlpha.  
The goal of this task was to convert low-fidelity wireframes into a high-fidelity user interface by adding visual design elements such as colors, typography, icons, and UI components.

The project focuses on designing a visually appealing and user-friendly interface for a mobile e-book reading application called **Bookspace**.

---

## Objective
The objective of this task is to:

- Transform wireframes into a high-fidelity interface
- Apply UI design principles
- Use colors, typography, and UI components effectively
- Create a visually engaging and user-friendly mobile app design

---

## App Concept
**Bookspace** is a mobile e-book reading application that allows users to discover, browse, and read digital books easily.  
The application is designed to provide a smooth reading experience with organized navigation and an attractive interface.

---

## Screens Designed
The high-fidelity UI design includes the following screens:

- Splash Screen
- Login / Welcome Screen
- Home Screen
- Book Library / Categories
- Book Details Screen
- Reading Interface
- User Profile Screen

Each screen is designed with proper UI elements and visual hierarchy.

---

## Design Elements
The design includes:

- Color palette
- Typography
- Buttons and UI components
- Icons and images
- Proper spacing and alignment

These elements help create a modern and user-friendly design.

---

## Tools Used
- Figma
- UI/UX Design Principles

---

## Design Approach
The design focuses on:

- Clean and modern interface
- Easy navigation for users
- Consistent visual style
- Improved readability for book content

---

## Learning Outcome
Through this task, I learned how to convert wireframes into high-fidelity UI designs by applying visual design principles and improving the overall user experience.
